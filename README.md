# Bot tribunal

## Fonctionnalité
- Fonction de tribunal qui permet de préserver vos serveur de tout abus de perm
- Totallement personnalisable
- Convient pour de gros comme de petit serveur

## Peu de dépedence requis
- discord.js
- chalk
- sqlite3

## Liste des commandes avec options
- **__Ban__**
    | ask
    | config
        | channel
        | juge
        | show
- **__bot__**
    | set
        | banner
        | color
        | name
        | pic
        | status
- **__reset__**
    | all
        | bot
        | serveur
    | channel
        | bot
        | juge
    | juge
        | bot
        | serveur
    | track
        | bot
        | channel
- **__trackrole__**
    | add
    | list
    | remove
- **__utils__**
    | help
    | ping
